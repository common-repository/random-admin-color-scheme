== Description ==

Random Admin color scheme - auto change your admin color scheme every time wp-admin page loaded.

If you works with your WordPress Admin everyday as we do at [WPArena](https://wparena.com "WordPress Arena"), it would be so boring to see a boring admin color scheme all day long.
This WordPress plugin will make your wordpress admin fresh in every page load. Make you even love WordPress move.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `random_admin_color_scheme` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
